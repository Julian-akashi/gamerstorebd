# Gamer Store – Proyecto WordPress ## Descripción del proyecto **Gamer Store** es una tienda en línea desarrollada con **WordPress + WooCommerce**, enfocada en la venta de productos relacionados con el mundo del gaming (videojuegos, consolas y perifericos). El objetivo del proyecto es mostrar el flujo completo de una tienda básica: - Catálogo de productos gaming. - Ficha de producto con imagen, precio y descripción. - Productos varibles y cupones - Carrito de compra. - Proceso de pedido / checkout. - Página de confirmación de pedido. Este repositorio contiene **todo el sitio de WordPress** necesario para ejecutar el proyecto en local, junto con la **base de datos SQL exportada**. --- ## Tema elegido - **Tema principal:** Astra. - **Constructor de páginas:** Elementor. - **Plugins principales:** - WooCommerce - Elementor - Smart Slider - Premium Addons La estética del sitio está orientada al **gaming**, usando imágenes y textos relacionados con videojuegos y productos gamer. --- ## Contenido del repositorio En la raíz del proyecto se incluyen los archivos estándar de una instalación de WordPress: - `wp-admin/` - `wp-content/` - `wp-includes/` - Archivos principales de WordPress (`index.php`, `wp-config.php`, etc.). - `database/gamerstorebd.sql` --- ## Requisitos previos Para ejecutar el sitio en local se necesita: - **Servidor local**: - XAMPP, WAMP, Laragon o similar.(en mi caso use XAMPP) - **PHP** (versión compatible con la versión de WordPress usada). - **MySQL / MariaDB** (para la base de datos). - Navegador web (Chrome, Firefox, Edge, etc.). En este README se asume el uso de **XAMPP** y que el directorio de proyectos es: ```text C:\xampp\htdocs\ ## Pasos para ejecutar el sitio en local Clonar o descargar el repositorio Clonar el repo con Git (o GitHub Desktop) en tu carpeta de proyectos, por ejemplo: C:\xampp\htdocs\gamerstorebd Iniciar servicios en XAMPP Abrir XAMPP. Iniciar Apache. Iniciar MySQL. Crear la base de datos en phpMyAdmin Ir a: http://localhost/phpmyadmin Hacer clic en Bases de datos. Hacer clic en Seleccionar archivo. Elegir el archivo:gamerstorebd.sql (que viene dentro del repositorio). Presionar Continuar para importar los datos.
